## 🚀 Как использовать

### Для пользователей

```bash
npm install popapi
popapi setup
popapi push ./files my-repo
```

### Для контрибьютеров

```bash
git clone https://github.com/ghudgjuggggb/popapi.git
cd popapi
npm install
npm start
```

Затем:
1. Создайте ветку для фичи
2. Пишите код
3. Откройте Pull Request
4. Дождитесь review

## 🌐 Публикация на GitHub

### Шаг 1: Создать репозиторий

```bash
git init
git add .
git commit -m "Initial commit: popApi v2.0"
git remote add origin https://github.com/ghudgjuggggb/popapi.git
git branch -M main
git push -u origin main
```

### Шаг 2: GitHub Settings

Откройте репозиторий на GitHub и:

1. Settings → About → Description
2. Settings → Topics → (добавьте: github, upload, cli, web, nodejs)
3. Settings → Features → Enable Discussions
4. Settings → Features → Enable Issues
5. Settings → Collaborators → Add maintainers

### Шаг 3: Badges (опционально)

В README добавить:
```markdown
[![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Node.js](https://img.shields.io/badge/node-%3E%3D12.0.0-green.svg)]
[![npm](https://img.shields.io/npm/v/popapi.svg)]
```
