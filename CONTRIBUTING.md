# Contributing to popApi

Спасибо за интерес к popApi! 🙏

Все приветствуются. Вот как помочь:

## Способы контрибьютить

### 1. Найти баг?
- Откройте Issue на GitHub
- Опишите что случилось
- Дайте пример для воспроизведения

### 2. Предложить идею?
- Откройте Discussion
- Объясните что хотите добавить
- Дайте примеры использования

### 3. Написать код?
- Fork репозиторий
- Создайте ветку: `git checkout -b fix/bug-name`
- Напишите код
- Создайте Pull Request

## Правила кода

✅ Чистый код без лишних комментариев
✅ Понятные имена переменных
✅ Тестируйте перед тем как пушить
✅ Обновляйте README если нужно

## Процесс Pull Request

1. Fork репозиторий
2. Создайте ветку (`git checkout -b feature/AmazingFeature`)
3. Commit изменения (`git commit -m 'Add AmazingFeature'`)
4. Push в ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Коммиты

Пишите понятные коммиты:

```
✅ Good
- "Fix file upload error"
- "Add GitHub webhook support"
- "Improve error messages"

❌ Bad
- "Fix stuff"
- "Update code"
- "Changes"
```

## Лицензия

Контрибьютя, вы соглашаетесь что ваш код будет под MIT лицензией.

## Вопросы?

- Откройте Issue
- Напишите в Discussion
- Проверьте документацию

Спасибо за помощь! ❤️
